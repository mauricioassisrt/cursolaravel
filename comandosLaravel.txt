migrations e model criando 
migrations possuem a estrutura da tabela os campos 

modell representa a entidade de uma tabela no banco de dados 

criando um novo model

após isso o projeto está pronto se ouver migrations é só rodar o comando para gerar a base de dados

após isso para inserir dados automaticamente, este comando insere todos os dados que ouver configurado
